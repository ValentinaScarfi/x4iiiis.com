Astral Insanity

Plot twist: The only insanity is the insanity that this drove me to.

The options menu can be navigated using up and down arrows or W and S, and allows the user to select a difficulty setting.


The game should be run as administrator for the highscore feature to work (although it is still a bit buggy). Also you're gonna get trapped if you open the high score thing from main menu cause I didn't implement a way back to the main menu... *facepalm.*

In game, the player can move using WASD or the arrow keys, and can shoot with the spacebar. Enemies spawn faster depending on the difficulty level and how many hits they can take also varies due to the user's preferred setting.

Powerups spawn in the top left corner. These include: Health Regen
Resupply
Weapons Upgrade
Decrease Enemy Speed (for a short time)

The game ends if the user collides with an enemy.
